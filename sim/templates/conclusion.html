Project Conclusion

This project demonstrates the simulation and evaluation of LEACH and AZC-PSO routing protocols in a wireless sensor network. Using the same node deployment for both protocols ensures fair comparison.

The final result depends on node placement and energy evolution, but in general AZC-PSO is expected to provide better balancing of transmission cost and cluster-head load, while LEACH remains simpler and historically important.

This website improves the presentation by separating theoretical discussion, live implementation, and result analysis into dedicated sections with professional visual design.
