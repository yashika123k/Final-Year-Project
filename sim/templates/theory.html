Conceptual Background

Theory

The scientific foundation behind the two routing protocols used in this project.

Wireless Sensor Network

A wireless sensor network consists of distributed sensor nodes that collect data, perform local communication, and transmit information to a base station with limited battery power.

LEACH Protocol

LEACH uses randomized cluster-head rotation to distribute communication energy across nodes, reducing repeated long-range transmission from the same sensors.

AZC-PSO Protocol

AZC-PSO applies zone-based rotation and more informed cluster-head selection using remaining energy, centroid proximity, and distance from the base station.

Why AZC-PSO can outperform LEACH

LEACH is simple and widely studied, but its random election can create uneven energy use. AZC-PSO improves this by selecting better cluster heads and managing relay behavior more intelligently, which can delay early node failure and improve total network lifetime.
